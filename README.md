# simulacion-resonador

Este repositorio contiene una simulación operativa de un resonador magnético
con agenda densa y demanda constante.

## Objetivo
Evaluar el funcionamiento del sistema bajo alta demanda,
considerando:
- Cambiador con tiempos variables
- Resonador como recurso crítico
- Agenda flexible basada en duración estimada de estudios

## Alcance actual
- Simulación de 1 día operativo
- Horario: 08:00 a 20:00
- Análisis determinístico + estocástico
- Sin Machine Learning (fase futura)

## Próximas fases
- Métricas operativas
- Simulación multi-día
- Incorporación de Machine Learning para aprendizaje de duraciones reales
